# Fantasy Football AI

This project trains a RL agent to nominate and bid on players in an ESPN fantasy football auction draft, with the goal to maximize team wins during the season.

Components:
- `ffai/src/ffai/auction_draft_simulator.py`: runs an auction draft simulation
- `ffai/src/ffai/rl_model.py`: RL model for training the agent.
- `ffai/src/ffai/season_simulator.py`: simulates a fantasy football season
